Q1: Why might we consider using Git?

Answer: We might consider using Git for its version management capabilities which make accessing older versions of software easy, and help teams collaborate on the same project for developing software.

Q3: What is the difference between Merging and Rebasing?

Answer: Merge combines branches by combining the hisotries of each branch, but rebasing adds one branch to the end of the other without combining the history.
